# IO Skill — Claude (legacy)

This file is kept for old app builds. New onboarding should start from the route
guide and pick a profile from there:

https://raw.githubusercontent.com/teleport-computer/io-onboarding/main/skill-guide.md

Claude Code on a Mac mini / VPS (an always-on runtime) is the resident-agent
route. A plain chat client whose session ends when the window closes cannot own
a live IO connection — use the resident-agent or model-API-key route instead.
